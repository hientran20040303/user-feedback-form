# Dynamic User Feedback & Interaction System

## 📌 Project Overview
This project is an interactive web-based feedback tool designed to demonstrate mastery of the JavaScript DOM and event-based programming. It captures user input in real-time, provides immediate visual feedback, and manages data flow using advanced event handling techniques such as delegation and propagation control.

The goal was to create a responsive user experience that validates data and updates the interface dynamically without requiring a page reload.

## 🚀 Technologies Used
*   **Language:** JavaScript
*   **Structure/Styling:** HTML5 & CSS3
*   **Core Concepts:** 
    *   Event Delegation & Bubbling
    *   DOM Manipulation
    *   Real-time Validation
    *   Event Propagation (`stopPropagation`)
    *   Keyboard and Mouse Event Listeners

## ✨ Key Features
*   **Live Character Tracking:** Uses `keydown` and `input` events to provide real-time character counts for user comments.
*   **Interactive UI Feedback:** Contextual tooltips appear on `mouseover` and disappear on `mouseout` to guide the user through the form.
*   **Robust Client-side Validation:** Prevents form submission if fields are empty, providing clear error messages.
*   **Efficient Event Management:** Implements **Event Delegation** to handle interactions from multiple input fields through a single parent listener, optimizing performance.
*   **Dynamic Content Injection:** Successfully submitted feedback is instantly appended to a display container without refreshing the browser.
